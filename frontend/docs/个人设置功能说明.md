# iFlytek多模态AI面试系统 - 个人设置功能说明

## 📋 功能概述

个人设置功能为用户提供了完整的个人信息管理和系统偏好配置，包括主题切换、字体大小调整、通知设置、账户安全等核心功能。

## 🎯 主要功能模块

### 1. 基本信息管理
- **头像上传**：支持JPG、PNG格式，最大2MB
- **个人资料编辑**：姓名、邮箱、手机号、职位、工作经验
- **技能标签管理**：多选技能领域标签
- **个人简介**：500字以内的自我介绍
- **表单验证**：实时验证邮箱格式、手机号格式等

### 2. 界面偏好设置
- **主题切换**：
  - 浅色主题（默认）
  - 暗黑主题
  - 跟随系统主题
  - 实时预览效果
- **字体大小调整**：
  - 小号字体（12px基准）
  - 中号字体（14px基准）
  - 大号字体（16px基准）
  - 全局生效，包括标题和正文
- **语言设置**：简体中文、繁体中文、English

### 3. 系统偏好配置
- **难度偏好**：初级、中级、高级
- **功能开关**：
  - 自动保存进度
  - 音效提示
  - 学习提醒
  - 面试提醒

### 4. 通知设置
- **邮件通知**：重要通知和更新邮件
- **短信通知**：紧急通知短信
- **浏览器推送**：桌面通知（需要权限）
- **专项通知**：
  - 面试邀请通知
  - 学习进度更新
  - 系统消息通知
  - 营销邮件（可选）
- **权限管理**：自动检测和请求浏览器通知权限

### 5. 账户安全
- **密码修改**：
  - 当前密码验证
  - 新密码强度检查
  - 确认密码匹配验证
  - 实时密码强度显示
- **两步验证**：增强账户安全性
- **登录设备管理**：查看和移除登录设备
- **会话超时设置**：自定义会话超时时间

### 6. 隐私设置
- **可见性控制**：
  - 个人资料可见性
  - 学习记录可见性
  - 面试记录可见性
- **数据控制**：
  - 数据分析授权
  - 第三方数据共享控制

## 🚀 使用指南

### 访问个人设置
1. 在系统中点击用户头像或设置图标
2. 选择"个人设置"菜单项
3. 或直接访问 `/personal-settings` 路径

### 主题切换操作
1. 进入"偏好设置"标签页
2. 在"界面主题"部分选择所需主题
3. 系统会实时预览主题效果
4. 点击"保存偏好设置"按钮保存

### 字体大小调整
1. 在"偏好设置"中找到"字体大小"选项
2. 选择小号、中号或大号字体
3. 页面会实时显示字体大小变化
4. 保存设置后全局生效

### 通知权限设置
1. 进入"通知设置"标签页
2. 开启"浏览器推送"开关
3. 如果浏览器提示权限请求，点击"允许"
4. 可以点击"测试通知"验证功能

### 密码修改流程
1. 进入"账户安全"标签页
2. 输入当前密码
3. 设置新密码（系统会显示密码强度）
4. 确认新密码
5. 点击"修改密码"完成

## 🎨 品牌一致性

### iFlytek品牌色彩
- 主色：#1890ff（iFlytek蓝）
- 辅色：#667eea、#0066cc、#4c51bf
- 暗黑主题下自动调整为高对比度版本

### 中文字体
- 优先使用Microsoft YaHei
- 备选字体：PingFang SC、Hiragino Sans GB、SimSun
- 确保中文字符清晰显示

### 交互设计
- 遵循Element Plus设计规范
- 保持操作一致性
- 提供清晰的状态反馈

## 🔍 故障排除

### 常见问题
1. **主题切换不生效**：
   - 检查浏览器是否支持CSS变量
   - 清除浏览器缓存重试

2. **通知权限被拒绝**：
   - 在浏览器设置中手动开启通知权限
   - 或使用其他通知方式

3. **设置无法保存**：
   - 检查浏览器是否禁用localStorage
   - 确认表单验证通过

4. **移动端显示异常**：
   - 检查viewport设置
   - 验证CSS媒体查询
